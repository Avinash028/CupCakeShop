# CupCakeShop

Steps to Start Frontend and Backend Server.

1) Clone the repo
2) Start the Backend

    i) cd CupCakeShop-master    
   ii) npm install   
  iii) cd Cup-Cake-Shop  
   iv) cd backend   
   v) node app.js
   
    
    
3) Start the Frontend

    i) open new terminal window    
   ii) cd CupCakeShop-master   
  iii) cd cup-cake-shop  
   iv) cd cup-cake-online   
   v) npm install      
   vi) npm start
